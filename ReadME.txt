READ ME

Berkshire Hathaway Console Application.

Functionality
Initializes with 3 reasons that I have chosen
Creates Database with startup argument when required.
Individual reasons may be added during runtime while tracking whose reason it is.
Bulk additions of reasons can be added from .txt files.
SQLite Database storage



Database Setup Instructions
Run application with argument 'init' to run DB setup
This will build the databases and add the initial reasons for working at Berkshire Hathaway
